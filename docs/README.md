### Getting started is very simple

```bash
# Install deps
$ yarn global add umi # or npm install -g umi

# Create application
$ mkdir myapp && cd myapp

# Create page
$ umi generate page index

# Start dev server
$ umi dev

# Build and deploy
$ umi build
```
